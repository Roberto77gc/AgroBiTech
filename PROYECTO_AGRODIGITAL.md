## 📱 **ESPECIFICACIONES TÉCNICAS MVP**

### **Funcionalidades Core**
```typescript
// Estructura de datos principal
interface Activity {
  _id: string;
  userId: string;
  date: string;
  cropType: string;
  surfaceArea: number;
  products: Product[];
  location: { lat: number; lng: number };
  totalCost: number;
  costPerHectare: number;
  notes?: string;
  weather?: WeatherData;
  photos?: string[];
}

interface Product {
  name: string;
  category: 'fertilizer' | 'pesticide' | 'seed' | 'other';
  quantity: number;
  unit: 'kg' | 'l' | 'g' | 'ml';
  pricePerUnit: number;
  supplier?: string;
}
```

### **APIs Necesarias**
- **Auth**: JWT + refresh tokens
- **CRUD**: Activities, products, users
- **Analytics**: Dashboard statistics
- **Export**: PDF/Excel generation
- **Upload**: Image handling (plants, receipts)

### **Performance Targets**
- **Load time**: <2s inicial, <500ms navegación
- **Mobile responsive**: 100% compatible
- **Offline capability**: Sync cuando vuelve conexión
- **SEO**: 95+ Lighthouse score
